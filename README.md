# Seefeld – Neue Tiroler Küche

Fullstack-Restaurant-Website mit Reservierungssystem.

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS, Framer Motion
- **Backend:** Python, FastAPI
- **Database:** Supabase (PostgreSQL)

## Setup & Local Development

### 1. Supabase Setup

1. Create a new project on [Supabase](https://supabase.com).
2. Go to SQL Editor and run:

```sql
create table reservations (
  id bigint generated by default as identity primary key,
  name text not null,
  phone text,
  email text not null,
  persons int,
  date date,
  time time,
  message text,
  created_at timestamp with time zone default now()
);

-- Enable RLS (Optional but recommended)
alter table reservations enable row level security;

-- Policy for inserting (allow anyone to make a reservation)
create policy "Enable insert for everyone" on reservations for insert with check (true);
```

### 2. Backend (Python/FastAPI)

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

Create `.env` in `backend/`:

```env
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_anon_key
FRONTEND_URL=http://localhost:5173
```

Run server:

```bash
uvicorn main:app --reload
```

### 3. Frontend (React)

Make sure you are in the root directory.

```bash
npm install
npm run dev
```

Create `.env` in root (if needed for build):

```env
VITE_API_URL=http://localhost:8000
```

## Deployment

### Backend (Render.com)

1. Create a new Web Service on Render.
2. Connect repository.
3. Root Directory: `backend`.
4. Build Command: `pip install -r requirements.txt`.
5. Start Command: `uvicorn main:app --host 0.0.0.0 --port $PORT`.
6. Add Environment Variables (`SUPABASE_URL`, `SUPABASE_KEY`).

### Frontend (Vercel)

1. Import project to Vercel.
2. Framework Preset: Vite.
3. Build Command: `npm run build`.
4. Add Environment Variable: `VITE_API_URL` (Set this to your Render Backend URL).

## Features

- **German Localization**: All text and formats tailored for DACH region.
- **Responsive Design**: Mobile-first approach using Tailwind.
- **Animations**: Premium feel using Framer Motion.
- **Data Persistence**: Reservations saved directly to Supabase via FastAPI.
